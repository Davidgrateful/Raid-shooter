# Onyix Raider

A fast-paced space shooter game built with Next.js and React.

## Tech Stack

- [Next.js](https://nextjs.org)
- [Tailwind CSS](https://tailwindcss.com)
- [TypeScript](https://www.typescriptlang.org)
- [JSFXR](https://github.com/grumdrig/jsfxr) for procedural audio

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- 13 enemy types with unique AI behaviors
- 5 powerup types (health, slow, fast shot, triple shot, pierce shot)
- Parallax star background
- Procedural sound effects
- Touch controls with dual virtual joysticks for mobile
- Keyboard + mouse controls for desktop
- Score tracking with local best score

## Controls

**Desktop:** WASD to move, mouse to aim and fire, M to mute, P to pause

**Mobile:** Left joystick to move, right joystick to aim and fire
